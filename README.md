# Who's That Pokémon?

A fan-made, browser-based game where you guess Pokémon from their silhouette

## How to Play

1. Select a Pokémon region (or leave as "All Regions").
2. Look at the silhouette of the Pokémon.
3. Type your guess in the input box and press **Guess** or **Enter**.
4. Use **Hint** if needed (up to 3 hints per Pokémon).
5. After a correct guess, the next Pokémon automatically loads.
6. Press **Give Up** to reveal the answer.
7. Press **Next** to skip to a new Pokémon at any time.
